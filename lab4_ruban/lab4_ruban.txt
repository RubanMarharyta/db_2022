Лабоаторна робота 4 Рубан Маргарити 

1)Встановлюємо NodeJS останньої версії та Angular з команди: npm install -g @angular/cli
2)У терміналі переходимо в директорію cd ..\lab4
3)Прописуємо команду: npm i cors --save
4)Прописуємо команду яка запустить підключення з віддаленою базою: nodemon index.js
5)Відкриваємо папку через термінал cd AngularApp
6)Пишемо команду для білдингу Angular: ng serve 
7)В браузері заходимо на http://localhost:4200/
